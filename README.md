# Musical Typing App

A beautiful web application inspired by Typatone that turns typing into music. Each keystroke produces a unique musical note, creating melodies as you type.

## Features

- 🎹 **Musical Keyboard**: Each key press plays a harmonious note
- 🌈 **Colorful Display**: Letters appear in vibrant colors
- ♪ **Visual Effects**: Floating musical note particles
- 🎵 **Multiple Scales**: Choose from Pentatonic, Major, Minor, Blues, or Chromatic scales
- 🔊 **Mute/Unmute**: Toggle sound on and off
- ✨ **Beautiful UI**: Gradient background with animated floating circles

## How to Use

1. Open `index.html` in your web browser
2. Start typing to create music!
3. Each keystroke will play a note and display a colored letter
4. Press Backspace to delete letters
5. Use the controls to:
   - Clear all text
   - Mute/unmute sounds
   - Change musical scales
